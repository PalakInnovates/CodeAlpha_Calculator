# CodeAlpha Internship Task

This repository contains the project completed as part of the CodeAlpha Internship Program.

## 🔹 Task: Build a Calculator 
This repository contains a fully functional and responsive Calculator Application developed as part of the CodeAlpha Internship Program.

The project demonstrates practical implementation of core front-end development concepts including structured layout design, styling, and interactive functionality using JavaScript.

### Features:
- Performs basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Interactive button-based input system
- Real-time result display
- Error handling for invalid inputs
- Clean and user-friendly interface
- Fully responsive design

## 💻 Technologies Used
- **HTML5** – Structure and layout  
- **CSS3** – Styling and responsive design  
- **JavaScript** – Application logic and interactivity
